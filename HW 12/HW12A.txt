Provide a brief definition of the following terms (in the JavaScript 
programming context):

 Parse - reading and converting some type of information into a 
designated format so it can be used within said format;

 Token - words or symbols used by code, which are reserved only
for coding and cannot be used to declare variable names;

 Delimiter - used to break strings apart within split();

 Class - templates for creating objects or special functions
for reading and changing data;

 Object - collection of properties, where a name/key is given to
an assigned value within an object;

 Property - part of an object that stores values and names;

 Method - functions stored as object properties;

 Prototype - the mechanism where objects inherit features from one another,
for example Array objects inherit from Array.prototype;

 JSON - format for storing and transporting data;

 fetch - brings a file with information over to a programmers code to be
able to use;
